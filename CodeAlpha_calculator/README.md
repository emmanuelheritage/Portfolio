# Simple Web Calculator


This repository contains the calculator that I built during my internship at CodeAlpha (February 20, 2026 to March 20, 2026.)


It is a basic web calculator built using **HTML, CSS, and JavaScript**.  
This calculator can perform basic arithmetic operations and provides a simple, responsive user interface.

## Live Demo

View the portfolio here:  
[🔗https://emmanuelheritage.github.io/CodeAlpha_tasks/CodeAlpha_calculator/index.html]

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear and delete functions
- Responsive design
